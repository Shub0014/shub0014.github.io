Cards

Card

A component that has two main sections; an image and a discription.
The card is an adaptation of panels, wells, and thumbnails from Bootstrap 3.
The .card-body deals with padding within the card component.
The .card-img-top is responsible for linking an image to the card component.
The .card-link attribute governs all links relating to the card by using anchors < a > tags.
